## Cara Menggunakan

1. Edit file `index.html` untuk mengubah informasi pribadi Anda:
   - Ganti "YOUR NAME" dengan nama Anda
   - Ubah job title sesuai pekerjaan Anda
   - Update semua link social media dengan username Anda

2. Tambahkan foto profil Anda:
   - Simpan foto profil Anda dengan nama `profile.png` di folder `assets/img/`
   - Atau ubah path di `index.html` sesuai lokasi foto Anda

3. Edit halaman Resume (`resume.html`):
   - Tambahkan pengalaman kerja Anda
   - Update pendidikan
   - Tambahkan skills yang Anda kuasai

4. Edit halaman Portfolio (`portfolio.html`):
   - Tambahkan project-project Anda
   - Update link ke project

5. Buka `index.html` di browser untuk melihat hasilnya

## Kustomisasi

### Mengubah Warna
Edit file `style.css` dan ubah warna gradient di bagian:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Mengubah Font
Edit bagian `font-family` di `style.css`

## Teknologi

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## Tips

- Pastikan semua link social media sudah benar
- Gunakan foto profil dengan kualitas baik (minimal 150x150px)
- Test website di berbagai ukuran layar
- Compress gambar untuk loading yang lebih cepat
